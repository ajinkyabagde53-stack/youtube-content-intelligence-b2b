# Grizzle — 3,623 SaaS YouTube Video Study

## Source
https://grizzle.io/blog/saas-youtube-study (May 2026)

## Study Overview
Analysed 3,623 videos across 69 SaaS YouTube channels. The most comprehensive data study on B2B SaaS YouTube performance published to date.

## Key Findings

| Finding | Data |
|---------|------|
| Creator-style thumbnails vs corporate-style | Creator-style earns **2.67x** median views |
| Optimal video description length | Under 10 words (single sentence) |
| Best thumbnail combination | Face + UI + text = **2x** baseline views |
| Shorts vs long-form on same channel | Shorts earn **48% more** views per video |
| Corporate/clean thumbnails (63% of dataset) | Earn *below* baseline — most common = worst performing |

## Implications for Content Brief Agent
- Thumbnail direction in every brief must specify: face + text + contrast color
- Description field should be capped at one sentence
- Recommend Shorts versions of every long-form video
- Avoid recommending corporate/clean thumbnail styles
